# Dataset para analisis y ML futuro

Esta etapa no implementa machine learning. Prepara datos exportables para analisis comercial y futuras pruebas.

Datasets:

- productos curados;
- historial de precios;
- resultados preview.

Variables utiles:

- fuente;
- titulo;
- URL;
- imagen;
- precio lista;
- precio transferencia;
- precio tarjeta;
- precio oportunidad;
- tipo de precio oportunidad;
- score comercial;
- flags de revision.

Antes de usar ML hace falta acumular datos historicos, revisar duplicados y mejorar calidad de URLs e imagenes.
