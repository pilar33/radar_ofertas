Ayuda del laboratorio web

El laboratorio permite pegar una URL de categoria o pagina de productos y ver rapidamente si el HTML expone datos utiles.

  1. Pega una URL concreta de categoria o listado.
  2. Revisa productos detectados por JSON-LD o heuristica HTML.
  3. Ajusta selectores CSS si hace falta.
  4. Guarda la configuracion como extractor reutilizable.
  5. Audita robots y terminos antes de automatizar.
  6. Procesa solo resultados seleccionados y con politica habilitada.
No se usa para scraping masivo, login, captcha, proxies ni Mercado Libre.
